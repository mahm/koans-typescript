# Koans w/TypeScript

TypeScriptを利用して解ける問題集

## Quick start

**node.jsのバージョンは >= 10が推奨です**

```
# clone repository
git clone git@github.com:mahm/koans-typescript.git

# change directory
cd koans-typescript

# yarn
yarn install

# check
node_modules/.bin/jest sample/fizzbuzz.test.js
```
